# 命令执行

## 常用Linux命令

| 命令     | 实际效果                   |
| -------- | -------------------------- |
| ls       | 列出当前目录               |
| cd       | 目录跳转                   |
| rm       | 删除                       |
| mv       | 重命名                     |
| whoami   | 查看当前用户               |
| uname -a | 列出内核版本等详细系统信息 |
| ifconfig | 查看当前机器内外网ip       |
| ping     | 判断目标机器是否可达       |
| touch    | 创建文件                   |
| mkdir    | 创建目录                   |

#### **find**

常用于在linux系统中查找某文件（支持通配符）的具体位置

```Plain
find / -name flag
find / -name f*
find / -name fla?
/为查找的起始根目录，以上命令会以/（根目录）为查找根目录，在其以及其子目录中查找符合后续命名规则的文件
```
